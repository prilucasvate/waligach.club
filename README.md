# 抽!!!
## 簡介
[抽!!!](https://waligach.club)
每天都在問「今天吃啥？」  
結果答案是「隨便」，「不要那個、也不要這個」  
熟悉吧？  
這就是為了終結這問題而誕生的。  

最核心的功能就是「**抽 今天吃啥**」：  
你丟選項、朋友也丟選項，  
大家在線上同一個房間，**同時**感受命運的力量。  
結果出來後，不用再吵、相信命運，因為都是隨機決定的，相信天意。


## 使用說明
* 在第一次進入時，需要先**填寫身分**(之後會記錄)  
* 進入後可以在**加個選項**中新增選項  
* 所有選項會出現在目前選項下方，可以按右邊的**X**刪除  
* **全部重設**會清空所有選項  
* 新增完選項後，可以透過**幫我選吧**來抽取結果  
* **快速抽**則會快速給出結果  
* **鎖定**會將目前抽獎按鈕先關閉  
* 在下方可以看到使用者的抽取紀錄  
* **倉庫**在左下紙箱按鈕，點擊進入倉庫可以儲存資訊，也可以加到資料夾中。   
* 再次點擊選項則會將選項加到目前選項。
